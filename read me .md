## 二、 项目简介 (Project Introduction)

本项目是一个基于 Python 全栈技术开发的本地化智能知识库助手。

针对通用大语言模型（LLM）在特定专业学科（如线性代数）中容易产生“知识幻觉”的问题，本项目采用了 **RAG** 技术架构。系统能够加载用户上传的本地 PDF 教材，对文档进行自动化切分与向量化存储，并结合 DeepSeek 大模型进行基于事实的精准答疑。

---

## 三、 核心功能模块 (Core Features)

### 1. 🧠 RAG 知识库引擎 (Backend)
* **文档解析**：利用 `PyPDFLoader` 读取本地 PDF 文档。
* **语义索引**：使用 `RecursiveCharacterTextSplitter` 进行文本切分，并调用 `HuggingFace Embeddings` 将文本转换为向量。
* **本地存储**：基于 `Chroma` 向量数据库实现本地化存储与检索，保障数据隐私。
* **精准问答**：构建检索链，让 AI 在回答前先“阅读”相关教材片段，确保答案严谨可靠。

### 2. 🎭 智能 Agent 工坊 (Interaction)
* **多角色切换**：内置“数学助教”、“编程导师”等预设专家模式。
* **自定义捏人**：用户可自由设定 AI 的昵称、头像（支持上传图片）及人设指令。
* **Meta-Prompting (元提示)**：集成了提示词优化算法。当用户输入简单指令（如“严厉的老师”）时，系统会自动调用 LLM 将其扩写为结构化、专业级的 System Prompt。

### 3. 🎨 沉浸式前端设计 (UI/UX)
* **莫兰迪视觉风格**：基于 Streamlit 框架，通过 CSS Injection 技术重构了原生界面，应用了低饱和度的莫兰迪配色方案。
* **Glassmorphism (毛玻璃)**：侧边栏与信息卡片采用磨砂玻璃拟态设计。
* **关键帧动画**：实现了聊天气泡的 `popIn` (弹性弹出) 与 `floating` (悬浮呼吸) 动画效果，提升交互的生命感。

## 四、 技术栈与文件结构 (Tech Stack)

* **开发语言**：Python 3.11
* **核心框架**：Streamlit (Web UI), LangChain (LLM 编排)
* **数据库**：ChromaDB (向量检索)
* **模型服务**：DeepSeek API
* **辅助库**：Pillow (图像处理), PyPDF (文档读取)

**项目文件结构：**

```text
My_AI_Assistant/
│

│
├── .streamlit/
```

## 五、 使用方法 (User Guide)

### 1. 环境配置
本项目依赖 Python 3.8+ 环境。首次运行前，请在终端（Terminal）进入项目文件夹并安装依赖：
```bash
pip install -r requirements.txt
```
### 2. 启动方式

本项目提供了两种启动方式，推荐使用方式一：

  * **方式一（推荐）**：直接双击项目根目录下的 **`启动助手.bat`** 脚本，系统将自动调用 Python 并在浏览器中打开应用。
  * **方式二（命令行）**：在终端运行以下命令：
    ```bash
    python -m streamlit run app.py
    ```

### 3\. 密钥配置说明

> **⚠️ 安全提示**：出于信息安全考虑，提交的代码包中 `secrets.toml` 配置文件内的 API Key 已置空。

程序已内置智能检测逻辑：若检测到配置文件为空，**会在运行界面的侧边栏自动弹出输入框**。请老师在运行界面手动输入 DeepSeek API Key 即可正常使用所有功能。
